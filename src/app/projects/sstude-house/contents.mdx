## 기여

- API 연동: axios 활용하여 데이터 요청 및 응답 처리

- 모바일 앱 연동: 인증번호를 이용한 헬스 데이터 연동 및 확인 페이지 구현

- Redux 상태 관리: 뷰티 및 헬스 데이터의 효과적인 상태 관리

- Python 서버 통신: 뷰티 진단 결과를 수신

- 달력 라이브러리 활용: 사용자가 일별로 데이터를 확인할 수 있게 하는 달력 라이브러리 활용

- 페이지 구현: 뷰티 및 헬스 데이터를 확인할 수 있는 페이지 구현

## 회고

- 스크럼의 중요성: 스크럼을 통해 팀원들과 현재 상황을 공유하면서, 어려움을 겪고 있는 사항이나 필요한 도움 등을 소통하는 과정에서 소통의 중요성을 깊이 느낌

- 팀 시너지의 중요성: 개인의 지식과 능력이 팀의 성장에 얼마나 큰 영향을 미치는지 체감했고, 이를 통해 우리 팀만의 독특한 방향성을 찾아가는 과정과 최적화된 방향성을 찾는 것이 중요하다는 것을 깨달음.
