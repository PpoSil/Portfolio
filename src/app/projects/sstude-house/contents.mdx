## 기여

[ 뷰티 (퍼스널 컬러 및 의상 관련) / 삼성 헬스 데이터 ]

API 연동: 모바일 앱 연동, 헬스 데이터 페이지 구성, Redux 상태 관리, Python 서버 통신, 달력 기능, 이전 결과 확인, 퍼스널 컬러 및 의상 진단 결과 페이지 구현

모바일 앱 연동: 인증번호를 이용한 헬스 데이터 연동 및 확인 페이지 구현

Redux 활용: 뷰티(퍼스널 컬러와 의상 진단) 및 헬스 데이터의 효과적인 상태 관리

Python 서버 통신: 뷰티 진단 결과를 수신

달력 기능: 사용자가 일별로 데이터를 확인할 수 있게 하는 달력 라이브러리 활용

이전 결과 확인: 선택한 일자에 대한 이전 결과를 볼 수 있는 페이지 구성

퍼스널 컬러 진단 결과 페이지: 퍼스널 컬러 진단 결과 확인 기능 구현

의상 진단 결과 페이지: 퍼스널 컬러 결과를 기반으로 한 의상 진단 결과 확인 기능 구현.
