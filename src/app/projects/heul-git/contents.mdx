## 주요 기능

---


### 메인 페이지 : 흘깃

인기 레포지토리 + 팔로우 중인 유저의 레포지토리를 [흘깃만의 알고리즘으로 큐레이션 서비스를 제공]

### 게시판 : 유레카

공유하고 싶은 이슈나 풀리퀘스트의 링크와 사진을 함께 첨부하여 글을 작성하는 유레카 게시판.

### 게시판 : 자유게시판

자유롭게 글을 작성하고 소통하는 게시판.

### 채팅 기능: GM

흘깃의 메세징 기능인 GM (Git Message).

### 프로필: 커밋 분석, 랭킹

사용자 프로필에서는 사용자의 커밋을 분석하고 팔로우한 유저들간의 랭킹을 시각화.

## 회고

---

- 처음으로 프론트엔드로서 다인 프로젝트에 참여.

- 우선순위 설정의 중요성: 필요한 작업을 우선적으로 처리하고, 이후에 추가적인 기능을 구현하여 작업의 효율성 높임.

- 코드 리뷰의 가치: 전공자 팀원의 코드 리뷰를 통해 코드를 개선하고, 더 효율적으로 작성하는 데 큰 도움이 된다는 것을 체감함.
