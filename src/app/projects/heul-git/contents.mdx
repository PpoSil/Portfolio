## ✜ 주요 기능


**메인 페이지 / 흘깃** : 사용자가 팔로우하거나 인기가 많은 레포지터리를 다양한 기준에서의 추천 알고리즘으로 큐레이션 서비스 제공.

---

**게시판 / 유레카** : 레포지터리 혹은,  관련된 정보( 이슈, 풀리퀘스트 등 )를 다른 사용자에게 공유할 수 있으며,  정보와 함께 사진과 텍스트를 작성하여 추가적인 정보를 전달 가능. 

---

**게시판 / 일반** : 커뮤니티 게시판 기능을 하며 자유게시판 외에도 다양한 카테고리에서 활동이 가능.

---

**실시간 대화** :  서로 팔로우한 유저들은 텍스트를 주고 받으며 대화 가능.

---

**프로필** : 사용자가 자주 쓰는 커밋을 분석하고, 팔로우한 유저들간의 랭킹을 시각화하여 제공.
