<summary>주요 기능 (차별점 추가할지 생각해보기) </summary>

### 메인 페이지 : 흘깃

유익한 Repository + Follow 중인 유저의 레포지토리를 흘깃만의 알고리즘으로 큐레이션 서비스를 제공

### 게시판 : 유레카

공유하고 싶은 Issue나 Pull Request의 링크와 사진을 함께 첨부하여 글을 작성하는 유레카 게시판.
댓글에서는 사용자 멘션 기능도 제공.

### 게시판 : 자유게시판

자유롭게 글을 작성하고 소통하는 게시판.

### 채팅 기능: GM

흘깃의 메세징 기능인 GM(Git Message).

### 프로필: 커밋 분석, 랭킹

사용자 프로필에서는 사용자의 커밋을 분석하고 팔로우한 유저들간의 랭킹을 시각화.
