# Have-It!

삶을 뜻하는 life와 기록을 뜻하는 로그의 합성어로 삶의 다양한 정보와 경험을 기록하는 행위를 뜻하는 라이프로깅을 메타버스에 적용하여 일상적인 라이프로깅 메타버스 서비스를 제작. 

모바일과 웨어러블, 웹 및 유니티의 집합체.

링크: https://github.com/Have-It/Have-It

---

## 기술 스택

프론트엔드 구현 - React, React.js, React-Query, styled-components

게임 - Unity 3D, C#

배포 - Nginx,Docker

협업 도구 - Figma, Jira, GitLab, Mattermost, Notion
